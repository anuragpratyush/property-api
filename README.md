
## Require Software
``````
1 - Composer
2-  PHP 7.4 or more

``````

## Project setup
``````
1 - composer install
2-  Make database changes in .env file

    DB_DATABASE= your database name
    DB_USERNAME=your database username
    DB_PASSWORD=your database password

3- php artisan migrate
4- php artisan serve

then the development server will start

``````
